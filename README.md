# Backend3
# Proyecto Backend III - Coderhouse

## Tecnologías
- Node.js
- Express
- MongoDB (Mongoose)
- JWT + Bcrypt
- Swagger
- Winston
- Zod
- Jest / Supertest

## Rutas principales
- POST /api/auth/register
- POST /api/auth/login
- GET /api/users/
- POST /api/adoptions/
- GET /api-docs (Swagger)

## Scripts
- `npm run dev`: Modo desarrollo
- `npm test`: Ejecutar tests

## Cómo usar
1. Clonar repo
2. Crear `.env` con las variables necesarias
3. `npm install`
4. `npm run dev`
